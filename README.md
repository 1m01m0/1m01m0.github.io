# GitHub Pages Homepage

这是一个可直接部署到 GitHub Pages 的静态个人主页。

## 修改入口

- `index.html`: 改名字、简介、项目、文章、邮箱和 GitHub 链接。
- `styles.css`: 调色板已按 Claude 官网 CSS 中的 swatch 变量整理，页面样式与动效只引用这些变量。
- `script.js`: 移动端导航、项目筛选、滚动显现、数字递增、首屏视差、Canvas 节点网络、终端打字、实时指标、磁吸按钮和 3D tilt。
- `assets/claude-workspace.png`: 主页首屏视觉图。
